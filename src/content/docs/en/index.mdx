## What you will experience in this workshop

Using **Claude Managed Agents**, you will create and run autonomous agents that operate in a cloud sandbox managed by Anthropic — no need to build the agent loop or the sandbox yourself. You can progress at your own pace: from the beginner track using only the console UI, to the intermediate track building with the CLI and Python SDK, and the advanced track tackling slide generation and automated news-site updates.
