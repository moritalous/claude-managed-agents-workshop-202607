## 開発を続けるためのツール

- **Claude Codeから始める** ・・・ 最新のClaude Codeで`/claude-api managed-agents-onboard`を実行すると、対話形式でエージェントのセットアップをガイドしてくれます
- **Anthropic CLI (`ant`)** ・・・ エージェントや環境をYAMLで定義してバージョン管理する運用に便利です。[CLIドキュメント](https://platform.claude.com/docs/en/api/sdks/cli)
- **SDKはPython以外にも** ・・・ TypeScript / Java / Go / C# / Ruby / PHPで同じことができます。[クイックスタート](https://platform.claude.com/docs/ja/managed-agents/quickstart)に全言語のコード例があります
