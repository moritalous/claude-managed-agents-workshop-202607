## ブログを書こう！

<Aside type="tip" title="参加後のお願い">
本イベントでは、参加後に**イベントに関連するブログ投稿**をお願いしています。書いたらXなどで`#ClaudeCommunityOsaka`を付けてシェアするか、運営まで教えてください。このページで紹介します。
</Aside>

「何を書けばいいか分からない」という人向けの、そのまま使える切り口です。

- **やってみた系** ・・・ 今日のハンズオンの再現レポート。詰まったところ・エラーとその解決こそが一番価値のある情報です
- **比較・考察系** ・・・ 「Messages APIで自作エージェントループを書く場合とManaged Agentsの比較」「Claude Code / Agent SDKとの使い分け」
- **応用系** ・・・ ハンズオンの発展。「調査結果をSlackに通知するエージェント」「自分の業務データで週次レポートを作るエージェント」など、自分の生活や仕事に寄せたネタは強い
- **深掘り系** ・・・ カスタムツール、MCPコネクタ、ファイル入力(PDF・画像)、アウトカム定義など、今日触れなかった機能を一つ選んで検証

## 掲載記事
